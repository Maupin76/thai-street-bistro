# Thai Street Bistro Website

## Overview

Welcome to the repository for **Thai Street Bistro**! This website was custom-built to showcase our authentic Thai cuisine, restaurant atmosphere, menu offerings, and online ordering options. Designed with a modern, mobile-first approach, the site provides an engaging experience for customers while highlighting the flavors and culture that make Thai Street Bistro unique.

Located in Mountain Home, Idaho, Thai Street Bistro combines traditional Thai recipes with a contemporary dining experience. This website serves as the digital home of the restaurant, allowing customers to explore the menu, learn about our story, view restaurant information, and connect with us online.

## Features

- Fully responsive design optimized for mobile, tablet, and desktop devices
- Modern user interface reflecting the Thai Street Bistro brand identity
- Dynamic menu rendering powered by JavaScript
- Organized menu categories for easy browsing
- Interactive navigation and user-friendly layout
- Accessible design with semantic HTML and descriptive image alt text
- Contact information, business hours, and location details
- Social media integration
- Clean and maintainable code structure following BEM methodology

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Maupin76/thaistreetbistro.git
```

2. Navigate to the project directory:

```bash
cd thaistreetbistro
```

3. Open `index.html` in your browser.

## Project Structure

```text
/images             -> Restaurant, menu, and branding images
/scripts            -> JavaScript files for site functionality
/styles             -> CSS files organized using BEM methodology
index.html          -> Main website entry point
README.md           -> Project documentation
```
